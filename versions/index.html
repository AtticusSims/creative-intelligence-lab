Design Variants

Creative Intelligence Lab — four directions to compare

Version A

Refined

Conservative improvement. Better typography (Space Grotesk + Inter), scroll animations, grain texture, horizontal scrolling gallery, polished layout.
Version B

Editorial

Magazine-style. Split hero with full-bleed painting, asymmetric grids, serif/sans contrast (Instrument Serif + DM Sans), cinematic project showcase, mix-blend-mode nav.
Version C

Immersive

Three.js particle network background, glass-morphism sections, interactive 360° panorama viewer, scroll-driven camera + color shifts. Fully experimental.
Version D

ASCII Immersive

Dense ASCII character grid driven by layered simplex noise, creating flowing organic patterns that morph over time. Scanlines, mouse-reactive glow, scroll-responsive dimming. Midjourney-inspired generative text field.

Original

Production Site

The original deployed version. Minimal CSS grid layout, scroll-reveal animations, panorama viewer, image gallery.
